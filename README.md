# ATTA_ProjetoFinal

Aula 02/05/17: combinamos com os professores de realizar o jogo Genius em python junto à uma plataforma online que conecte os jogares, 
para que possam jogar entre si (duelo) e para que estejam rankeados (high score de quem conseguiu lembrar sequências maiores). Também,
criar versões do jogo alteradas para aumentar complexidade; por exemplo, um genius com mais de 4 cores (6, 8 ou 10) ou um que mude a
posição de cada uma das cores (para que o usuário lembre a cor e não a posição).
Para semana que vem: trazer uma versão funcionando do genius básico e algo relacionado a plataforma online.

Aula 09/05/17: combinamos com os professores de realizar para a aula que vem finalizar o jogo básico em tkinter funcionando. Se possível,
realizar o jogo em pygame.
